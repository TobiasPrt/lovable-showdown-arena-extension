# Lovable Dev Prompt Extension

## Installation

1.  Go to the [Releases](https://github.com/lovable-dev/lovable-dev-prompt-extension/releases) page.
2.  Download the latest `.zip` file.
3.  Extract the contents to a directory.
4.  Go to `chrome://extensions` in your Chrome browser.
5.  Enable Developer Mode.
6.  Click "Load unpacked".
7.  Select the directory containing the extracted extension files.
8.  Refresh the page you want to use the extension on.

The extension should now be loaded and active.
